# Contributing to Fundbox

Thank you for your interest in contributing to Fundbox! This document provides guidelines and instructions for contributing.

## Getting Started

1. Fork the repository
2. Clone your fork: `git clone https://github.com/yourusername/fundbox-site.git`
3. Create a new branch: `git checkout -b feature/your-feature-name`
4. Make your changes
5. Commit your changes: `git commit -m "Add some feature"`
6. Push to your fork: `git push origin feature/your-feature-name`
7. Open a Pull Request

## Development Setup

1. Install dependencies: `npm install`
2. Copy environment variables: `cp .env.example .env.local`
3. Run the development server: `npm run dev`

## Code Style

- Follow the existing code style
- Use TypeScript for all new code
- Run `npm run lint` before committing
- Ensure all tests pass (if applicable)

1. Update the README.md with details of changes if applicable
